# 💡 Concepts

A collection of technical concepts explored in depth — each folder is dedicated to deeply understanding a specific idea, algorithm, or technology through notes, diagrams, and minimal code.

## Structure

```
concepts/
└── topic-name/
    ├── README.md       # What it is, why it matters, mental model
    ├── src/            # Minimal illustrative code
    ├── notes.md        # Deep dive notes
    └── references.md   # Books, links, further reading
```
